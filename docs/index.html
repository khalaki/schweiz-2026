База Naters / Belalp — канатка буквально над хатою. Тиждень денних хайків (Алеч, Цермат), потім hut-to-hut: Europahütte → Zermatt → Cabane des Dix (навпроти льодовика Cheilon). Без намета — лише легальні ночівлі в хатинах.
Липень у Вале = +30 °C у долині Рони і ~0…+5 °C зранку на 2800–3000 m. Мембрана обовʼязкова: грозові комірки майже щодня після обіду. Старт ранній на гребені.

Дорога · 30.06 – 1.07

30.06 · Kraków → Liechtenstein

  • ~1050 км, 10–11 год. Найшвидше: Чехія → Австрія (Brno → Wien → Salzburg → Innsbruck → Feldkirch).
  • Кава в горах: канатка Malbun або Gaflei + короткий трейл на Fürstensteig — видовий, без довгого набору.

1.07 · Liechtenstein → Naters

  • ~230 км, 4–5 год чистої їзди. Chur → Andermatt → Realp → Furka Pass → Gletsch → Naters.
  • Перевал відкритий у липні — безпечний план.
  • Belvédère + Ронський льодовик + льодова печера (~9 CHF вхід).
  • Опційно з Gletsch: Grimsel (+15 хв) заради видів на водосховища — і назад.

Спорядження · базовий набір на особу

Верх · шари
  • 2–3 синтетика/меринос (не бавовна)
  • Легкий фліс (grid / R1)
  • Пухова/синтетика ~200–300 g — вечори в хатинах, Cabane des Dix (~2928 m)
  • Мембрана обовʼязково — грози, Bettmerhorn–Eggishorn без схову
  • Шапка + баф + легкі рукавички (Pas de Chèvres, металеві сходи)
  • Кепка, окуляри cat. 3, SPF 50
Низ · ноги
  • Трекінгові штани / софтшел + шорти
  • Легінси/термоштани — холодний ранок на перевалах
  • Дощові штани — легкі, якщо є
  • Черевики mid або trail з протектором (T2–T3, не альп-бути)
  • 3–4 шкарпетки + суха «свята» пара
  • Сандалі/крокси для хатин (своє взуття знімають)
Хатини · 3 ночівлі
  • Вкладиш у спальник (liner) — обовʼязковий у швейцарських хатинах (~100 g)
  • Беруші
  • Налобний ліхтар (світло гасять ~22:00)
  • Палиці — спуск з Europaweg, Pas de Chèvres; особливо без досвіду довгих спусків
  • Готівка CHF ~250/особа на hut HB ×3 + дрібниці
Логістика · авто
  • 30.06: довгий трансфер, ночівка Liechtenstein або по дорозі.
  • 1.07: Furka Pass — Belvédère, льодова печера, опц. Grimsel.
  • Цермат: лише Täsch (поїзд), авто не в село.
  • 8–10.07: авто Randa → 10.07 ранок поїзд до Randa → Arolla → Dix.
  • 11.07: розумний виїзд — ночівка центральна Швейцарія → 12.07 ~9 год до Kraków.
